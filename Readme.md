# CustoMall: Backend

Build a customizable e-commerce platform with a seamless user experience.


## Tech Stack

**Backend:** Node, Express, MongoDb

**Frontend** React.js, Tailwind CSS, 

## .env

**PORT** = 5000

**MONOG_URL** = mongodb+srv://CustoMall:****@customall***********

**JWT_SECRET** = mysecretkey

# Setup Backend 

**npm install**

**npm start**

# Frontend Setup
**yarn install**

**yarn run dev**
